# Breakthrough Author Live

A premium landing page for **Breakthrough Author Live** — a 4-week live challenge that helps coaches write and launch a bestselling book.

## Live Site

[https://mehak-afk.github.io/breakthrough-authorlive/](https://mehak-afk.github.io/breakthrough-authorlive/)

## Pages

- **Landing Page** (`index.html`) — Full sales page with hero, program timeline, testimonials, pricing, and more
- **About Page** (`about.html`) — Meet the founder, Mehak Bhatia

## Features

- Dark / Light mode toggle (persisted in localStorage)
- Scroll-triggered reveal animations using IntersectionObserver
- Countdown timer for next cohort
- Sticky navbar with backdrop blur on scroll
- Gold shimmer hover effect on CTA buttons
- Fully responsive — mobile-first design
- Accessible — ARIA labels, keyboard navigation, `prefers-reduced-motion` support
- Grain texture overlay for luxury editorial feel

## Design

- **Palette:** Navy (`#0D1B2A`), Cream (`#F7F2E8`), Gold (`#C9A84C`)
- **Fonts:** Cormorant Garamond (display), Jost (body) via Google Fonts
- **Style:** Luxury editorial — clean typography, generous whitespace, subtle animations

## Tech Stack

- HTML5 (semantic markup)
- CSS3 (custom properties, animations, grid, flexbox)
- Vanilla JavaScript (no frameworks or dependencies)
- Hosted on GitHub Pages
